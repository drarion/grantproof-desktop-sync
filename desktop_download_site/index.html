GrantProof Desktop Sync

Le compagnon Windows qui reçoit les preuves collectées sur mobile dans un dossier local GrantProof. Vous pouvez ensuite sauvegarder ou synchroniser ce dossier avec vos outils habituels, sans stockage GrantProof.

Parcours recommandé

  1. Installez GrantProofDesktopSync.exe sur le PC cible.
  2. Choisissez le dossier local GrantProof.
  3. Affichez le QR de jumelage côté PC.
  4. Scannez-le depuis l’application mobile.

Publication propre

Cette page sert de modèle simple. Publiez de préférence la page docs/index.html du repo desktop avec GitHub Pages afin de connecter automatiquement le bouton de téléchargement aux releases du dépôt.

Repo recommandé : grantproof-desktop-sync / docs/index.html

Installation mobile

Dans l’application GrantProof, le flux principal est maintenant : installer le compagnon PC, configurer le téléphone, puis jumeler avec le QR du PC.
