Télécharger pour Windows Voir les releases
Le bouton de téléchargement pointe vers la dernière release publique publiée par le workflow Windows. Si le téléchargement n’est pas encore disponible, ouvrez la page des releases puis relancez un build GitHub Actions pour publier la release desktop-latest.

Installation en 3 étapes

Le parcours recommandé est très simple : ouvrez cette page sur le PC, installez le compagnon, puis scannez son QR dans GrantProof Mobile.

1
Ouvrez cette page sur votre PC
Depuis l’ordinateur, ouvrez cette adresse dans votre navigateur pour accéder au téléchargement du compagnon Windows.
2
Téléchargez puis lancez le compagnon
Téléchargez GrantProofDesktopSync.exe, ouvrez-le, choisissez le dossier local, cliquez sur Enregistrer puis sur Démarrer.
3
Scannez le QR depuis GrantProof Mobile
Quand le QR s’affiche sur le PC, revenez dans l’application mobile pour jumeler le téléphone et activer la synchronisation locale.

Pourquoi le compagnon PC ?

GrantProof Desktop Sync centralise les preuves sur le poste du client, prépare le dossier local GrantProof et régénère automatiquement les rapports premium à chaque synchronisation.

Local-first
Les fichiers restent côté client, sur le PC ou le réseau de l’organisation. Pas de dépendance à un stockage externe pour récupérer les preuves terrain.
Rapports premium
Les rapports Word et Excel donor-friendly sont régénérés automatiquement avec annexes médias, galerie images et registre projet enrichi.
Jumelage simple
Un QR compact ou un code manuel permet de relier rapidement le téléphone au bon poste de travail, sans complexité technique.
