GrantProof Desktop Sync

Le compagnon PC de GrantProof

Recevez les synchronisations mobiles dans un dossier local GrantProof, sans serveur GrantProof et sans complexité Microsoft ou Google côté terrain.

Télécharger pour Windows Voir les releases

Installation

Téléchargez le fichier GrantProofDesktopSync.exe, lancez-le, choisissez votre dossier local et démarrez le serveur local.

Jumelage

Le logiciel affiche un QR code unique. L’agent le scanne dans GrantProof Mobile pour lier le téléphone au bon ordinateur.

Synchronisation

Les fichiers arrivent dans votre dossier local GrantProof. Ensuite, vous pouvez les sauvegarder ou les synchroniser avec vos outils habituels.
