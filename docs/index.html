GrantProof Desktop Sync • v9.3.0

Le compagnon PC premium de GrantProof

Recevez les synchronisations mobiles dans un dossier local GrantProof, générez des rapports Word et Excel premium et gardez les données côté client.

Télécharger pour Windows Voir les releases
Activez GitHub Pages depuis ce dépôt pour connecter automatiquement cette page aux GitHub Releases du repo.

Installation

Téléchargez le fichier GrantProofDesktopSync.exe, lancez-le, choisissez votre dossier local et démarrez le serveur local.

Jumelage

Le logiciel affiche un QR code unique. Le membre du staff le scanne dans GrantProof Mobile pour lier le téléphone au bon ordinateur.

Synchronisation

Les fichiers arrivent dans votre dossier local GrantProof. Le compagnon régénère aussi des rapports premium Word et Excel donor-friendly, avec annexes médias et registre enrichi, puis vous pouvez sauvegarder ou synchroniser ce dossier avec vos outils habituels.
