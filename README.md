# GrantProof Demo Starter

GrantProof is a premium Flutter demo for NGOs and grant-funded projects.

This V3 demo includes:
- FR / ENG language toggle
- editable projects with donor, country, activities and outputs
- evidence capture with up to 5 photos
- manual location text + GPS coordinates
- story capture
- local-first demo persistence
- collaborative workspace demo screens for:
  - Google Shared Drive
  - Microsoft 365 / SharePoint
- sync center UI with simulated offline queue and sync flow
- donor-ready PDF demo export

## Important
This repository is still a **demo / prototype**:
- it does **not** upload any data to GrantProof servers
- it does **not** implement real OAuth connectors yet
- the workspace connection and sync are **simulated in local storage** for product validation
